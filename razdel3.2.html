Раздел 3: Бокс-модель и позиционирование элементов.

Знакомство с фронтенд-разработкой и её ролью

Бокс-модель и позиционирование элементов являются важными концепциями в CSS, которые определяют, как элементы отображаются и взаимодействуют друг с другом на веб-странице.
1. Бокс-модель:
Бокс-модель представляет собой структуру каждого HTML-элемента в виде прямоугольного "бокса", включающего внутреннее содержимое, отступы, границы и поля. Основные компоненты бокс-модели:
• Content (Содержимое): Фактическое внутреннее содержимое элемента, такое как текст или изображение.
• Padding (Отступы): Пространство между содержимым и границей элемента.
• Border (Граница): Линия, окружающая внутреннее содержимое и отступы элемента.
• Margin (Поля): Пространство вокруг границы элемента, определяющее расстояние между этим элементом и соседними элементами.
Пример CSS-правила для применения бокс-модели:

.box { width: 200px; height: 150px; padding: 20px; border: 2px solid #333; margin: 10px; }
Это правило устанавливает размеры, отступы, границу и поля для элемента с классом .box.
2. Позиционирование Элементов:
Позиционирование элементов в CSS определяет, как элемент размещается на странице относительно других элементов. Основные значения свойства position:
• Static (Статическое): Элемент размещается в потоке документа, игнорируя свойства top, right, bottom и left.
• Relative (Относительное): Элемент сдвигается относительно своего нормального положения в потоке документа.
• Absolute (Абсолютное): Элемент позиционируется относительно ближайшего позиционированного (не статического) родителя. Если такого нет, элемент позиционируется относительно документа.
• Fixed (Фиксированное): Элемент фиксируется относительно видимой области окна браузера.
• Sticky (Липкое): Элемент "липнет" к верхней или нижней границе родительского контейнера в зависимости от прокрутки.
Пример CSS-правила для позиционирования элемента:
.header { position: fixed; top: 0; left: 0; width: 100%; background-color: #333; color: #fff; }
Это правило делает элемент с классом .header фиксированным вверху страницы с черным фоном и белым текстом.
