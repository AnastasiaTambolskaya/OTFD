Раздел 4: Введение в JavaScript: переменные, типы данных, операторы.

Знакомство с фронтенд-разработкой и её ролью

JavaScript является высокоуровневым языком программирования, широко используемым для создания интерактивных веб-приложений. Введение в этот язык включает в себя понимание переменных, типов данных и операторов.
1. Переменные:
Переменные в JavaScript служат для хранения и обращения к данным. Они объявляются с использованием ключевого слова let, const или var. Примеры:
let age = 25; // Используется для изменяемых значений const pi = 3.14; // Используется для неизменяемых значений var name = "John"; // Старый способ объявления переменных
2. Типы Данных:
JavaScript является динамически типизированным языком, что означает, что тип данных переменной может изменяться в процессе выполнения программы. Основные типы данных включают:
• Числа (Numbers): Целые числа и числа с плавающей запятой.
• Строки (Strings): Текстовые данные, заключенные в кавычки.
• Булевы значения (Boolean): true или false.
• Массивы (Arrays): Упорядоченные списки данных.
• Объекты (Objects): Коллекции ключ-значение.
let num = 42; let greeting = "Привет, мир!"; let isTrue = true; let fruits = ['яблоко', 'груша', 'апельсин']; let person = { name: 'John', age: 30 };
3.Операторы:
Операторы в JavaScript выполняют операции над данными и переменными. Некоторые основные операторы включают:
• Арифметические операторы: +, -, *, /, %.
• Операторы сравнения: ==, !=, >, <, >=, <=.
• Логические операторы: && (и), || (или), ! (не).
• Операторы присваивания: =, +=, -=, *=, /=.
let sum = 5 + 10; // Арифметическое сложение let isEqual = (2 * 3) == (7 - 1); // Оператор сравнения let logicalResult = true && false; // Логическое "и" let assignmentExample = 10; // Оператор присваивания
Изучение переменных, типов данных и операторов в JavaScript - ключевой шаг для освоения этого языка программирования и создания интерактивных веб-приложений.
