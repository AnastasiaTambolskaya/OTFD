Раздел 4: Взаимодействие с DOM и событиями.

Знакомство с фронтенд-разработкой и её ролью

DOM (Document Object Model) представляет собой структуру документа в виде дерева объектов, которая может быть изменена с использованием JavaScript. Взаимодействие с DOM и обработка событий играют ключевую роль в создании динамичных и интерактивных веб-приложений.
1. Взаимодействие с DOM:
JavaScript используется для манипуляции элементами на веб-странице через DOM. Это позволяет изменять содержимое, стили, атрибуты и структуру страницы динамически. Примеры взаимодействия с DOM:
// Получение элемента по идентификатору let element = document.getElementById('myElement'); // Изменение содержимого элемента element.innerHTML = 'Новый текст'; // Изменение стилей элемента element.style.color = 'blue'; // Создание нового элемента let newElement = document.createElement('p'); newElement.textContent = 'Новый параграф'; // Добавление элемента в DOM document.body.appendChild(newElement);
2. Обработка Событий:
События происходят в ответ на действия пользователя или изменения веб-страницы. JavaScript позволяет создавать обработчики событий для реагирования на эти события. Примеры обработки событий:
// Добавление обработчика клика element.addEventListener('click', function() { alert('Элемент был кликнут!'); }); // Добавление обработчика изменения значения в поле ввода let inputElement = document.getElementById('myInput'); inputElement.addEventListener('change', function(event) { console.log('Новое значение: ' + event.target.value); });
3. Пример взаимодействия с DOM и событиями:
DOM let headerElement = document.getElementById('myHeader');
let buttonElement = document.getElementById('myButton');
Взаимодействие с DOM и обработка событий делают веб-приложения более динамичными и отзывчивыми, обеспечивая богатый пользовательский опыт.
