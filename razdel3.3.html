Раздел 3: Применение стилей к тексту, изображениям, фонам.

Знакомство с фронтенд-разработкой и её ролью

В веб-разработке стилизация элементов играет важную роль в создании привлекательных и удобочитаемых веб-страниц. Применение стилей к тексту, изображениям и фонам осуществляется с использованием различных свойств CSS, что позволяет разнообразно оформлять контент на сайте.
1. Стилизация Текста:
• Цвет текста: Свойство color определяет цвет текста. Например:
p { color: #333; }
• Шрифт: Свойства, такие как font-family, font-size и font-weight, позволяют настраивать шрифт текста.
h1 { font-family: 'Helvetica', sans-serif; font-size: 24px; font-weight: bold; }
• Текстовые эффекты: Различные свойства, такие как text-decoration и text-transform, позволяют добавлять подчеркивание, зачеркивание или изменять регистр текста.
2. Стилизация Изображений:
• Размер и обтекание: Свойства width и height определяют размер изображения. float позволяет задать его обтекание текстом.
img { width: 200px; height: 150px; float: left; }
• Круглые углы: Свойство border-radius создает закругленные углы для изображения.
img { border-radius: 10px; }
3. Стилизация Фонов:
• Цвет фона: Свойство background-color задает цвет фона элемента.
body { background-color: #f8f8f8; }
• Фоновое изображение: Свойство background-image позволяет установить изображение в качестве фона.
header { background-image: url('header-bg.jpg'); background-size: cover; }
• Повторение фона: Свойство background-repeat контролирует повторение изображения по горизонтали и вертикали.
section { background-image: url('pattern-bg.png'); background-repeat: repeat-x;
Применение стилей к тексту, изображениям и фонам в CSS предоставляет разработчикам широкий спектр возможностей для творческой стилизации веб-содержимого, делая страницы более интересными и функциональными.
